# Hypervisor detector on Cisco Nexus switches
***

## hyperdetect.py

### Description:
Скрипт, для получения списка свободных IP в заданой подсети.

### Usage
    hyperdetect.py -h                   - help menu
    hyperdetect.py -a                   - run scipt
    hyperdetect.py -i                   - device info
